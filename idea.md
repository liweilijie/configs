# 改键

macOs copy

- Find Usages: ctl+\, s
- Back: ctl+t, t
- Search Everywhere: ctl+\, t
- Previous Method: cmd+[
- Go to Declartion or Usages: ctl+]
- Select Tab #1 #2 #3 设置成 `cmd+[1234]`
